## 使用方法

### 方式1：使用VS Code（推荐）

1. 安装VS Code（Apple Silicon版本）
2. 安装推荐插件（C/C++、Code Runner等）
3. 打开项目文件夹
4. 使用F5编译运行

### 方式2：使用终端

1. 打开终端
2. 进入代码示例目录
3. 运行编译脚本：`./编译所有示例.sh`
4. 执行程序：`./程序名`

### 方式3：使用Xcode

1. 安装Xcode
2. 创建Command Line Tool项目
3. 复制代码到main.c
4. 点击运行按钮
