## 编译和运行

### 使用GCC编译器（推荐）

```bash
# 编译
gcc-14 文件名.c -o 输出文件名

# 运行
./输出文件名
```

### 使用Clang编译器（macOS自带）

```bash
# 编译
clang 文件名.c -o 输出文件名

# 运行
./输出文件名
```

### 示例：编译并运行Hello World

```bash
gcc-14 第2章_hello.c -o hello
./hello
```

## 常见问题

### 1. gcc命令未找到

```bash
# 使用完整路径
/opt/homebrew/bin/gcc-14 文件名.c -o 输出文件名

# 或创建别名
echo 'alias gcc="gcc-14"' >> ~/.zshrc
source ~/.zshrc
```

### 2. 编译权限问题

```bash
chmod +x 输出文件名
```

### 3. 头文件找不到

```bash
# 安装Xcode命令行工具
xcode-select --install
```

## 学习建议

1. **循序渐进**：按照章节顺序学习，不要跳过基础内容
2. **多动手**：每章的代码都要亲自编写和运行
3. **理解原理**：不要死记硬背，理解每个概念的原理
4. **做练习**：完成每章的课后练习，巩固所学知识
5. **调试代码**：学会使用调试工具，找出程序错误
