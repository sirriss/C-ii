# C语言零基础教程视觉设计方案

## 1. 设计理念

### 整体视觉方向

- 以清晰、结构化、富有科技感的视觉风格，打造一份专为M1 Mac用户设计的C语言学习指南。设计参考Apple Developer Documentation的简洁美学与MIT Technology Review的排版清晰度，旨在将复杂的编程概念转化为直观、易于理解的学习模块，强调在Apple Silicon设备上的原生学习体验。

### 内容密度

- 中高密度。专为教程和文档设计，每页聚焦3-5个核心知识点或代码块，确保信息清晰、易于消化。在代码示例和解释文本之间取得平衡，留白约15-20%，避免信息过载，提升学习效率。

## 2. 色彩系统

**主色调**

- 背景色：#1C1C1E（深灰黑色，营造专业编程环境）
- 主色：#0A84FF（Apple系统的科技蓝，用于标题和重要元素）
- 辅助色：#8A8A8F（中性灰，用于次要信息）
- 强调色：#FF9F0A（橙色，用于突出关键代码和提示）

**文本颜色**

- 深色文本（用于浅色背景）：#1D1D1F
- 浅色文本（用于深色背景）：#F2F2F7

## 3. 字体设计

**标题字体**

- 中文字体
  - Chiron Hei HK（现代几何黑体，线条锐利，科技感强，匹配C语言的精确性）
- 英文字体
  - Hedvig Letters Sans（个性鲜明，具有现代技术感，为标题增添独特的品牌气质）

**正文字体**

- 中文字体
  - Noto Sans SC（风格中性，结构规范，确保教程内容在任何屏幕上都清晰易读）
- 英文字体
  - Quattrocento Sans（经典无衬线体，温和易读，在小字号下依然表现出色，适合代码注释）

## 4. 图像风格

- 使用简洁的代码截图，并配合标注和箭头进行说明。可搭配抽象的几何图形、简洁的图标（如终端、文件夹、代码块）以及Apple风格的界面截图，避免使用真实照片，保持整体的现代感和技术美学。
